SnakeAI

It is a Snake game implemented using Python and the Pygame library for graphics and user input. In addition to the traditional gameplay, I integrated an AI player using the Torch library, where I implemented a deep reinforcement learning algorithm to enable the AI to play the game on its own. To efficiently manipulate and process data, I utilized the Numpy library. Furthermore, I employed the Matplotlib library to visualize the data, which enhanced the understanding of the AI's performance during gameplay.

Technologies: Python, Pygame, Torch, Numpy, and Matplotlib

Graph App

GraphApp is a web-based graph visualization tool that I developed. The project utilizes Python, Flask, and SocketIO on the backend, along with JavaScript and HTML on the frontend. The application generates a randomized matrix to create a graph, which is then displayed on a canvas. Users have the ability to interact with the graph by moving vertices to their desired positions.

Technologies: Python, Flask, SocketIO, JavaScript, HTML, Heroku, gunicorn

React Movie Database App

React Movie Database App is a web application built using React that utilizes a movie database API to display information about movies. The app allows users to explore the most popular movies and provides various filter options to customize the movie list based on type, release date, and popularity. Additionally, users can add movies to their watch list and favorites, which are saved in the local browser storage for easy access. The app offers an interactive and user-friendly interface, providing an enjoyable movie browsing experience for users.

Technologies: React, JavaScript, HTML, CSS, Movie Database API
